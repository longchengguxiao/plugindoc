### 青海之行（我的视角）

#### 第一天！

> 2023年4月25日，在两周的精密策划后，我们坐夜车前往西宁
> 
> 幸好买了脖枕，不然睡觉都不舒服

![29](/#/recourse/29.jpg ':size=50%')

![30](/#/recourse/30.jpg ':size=50%')

> 在火车上虽然有13个小时的硬座车程，但是有她我一点也不无聊，一起吃着泡面看着狄仁杰讨论剧情，时间过得额外的快

![31](/#/recourse/31.jpg ':size=50%')

> 我们还带了一个枕头，可以打开成一个小杯子，这就让我们丹丹宝在车上也能睡得很舒服

![32](/#/recourse/32.jpg ':size=50%')

> 租了一辆车，捷达VS5，学完驾照后的第一次开车说实话还是有点紧张的

![33](/#/recourse/33.jpg ':size=50%')

> 吃羊肠面咯！是一位美食博主推荐的地方，面很好吃，把小丹宝都辣哭了嘿嘿（当然我也被辣哭了）
> 
> 而且价格也很实惠，一大碗只要14元。
> 
> 但是有一个小插曲，第一次来的时候是下午4点，在酒店收拾了一会儿房间休息了一下，结果到地方人家告知我们3点半就卖完了下班了，早上6点开始营业。
> 
> 于是我们就第二天早上6点起床来吃的做早饭，不得不说还是挺值得，很美味。
